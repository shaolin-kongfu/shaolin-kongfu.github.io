为我的博客更换主题

发表于 2020-02-26 | 分类于 博客 |

前面介绍了如何搭建一个hexo博客，今天学习一下为博客更换主题

更换主题步骤

选择一个合适的主题：

经过笔者的多次比较，最终决定使用NEXT主题，这个主题是台湾同胞写的，并且比较火，如果使用过程出现问题，网上已经有很多答案

安装主题

  1. 打开之前自建的博客文件夹，运行git窗口
  2. 输入： （安装过程可能比较慢，甚至是失败，多运行几次就好了）
    也可以自己上网搜索next主题压缩包，解压到自建博客文件夹里的themes文件夹

启用主题

  1. 在自己的博客文件夹里打开_config.yml，使用搜索功能搜索“themes:landscape”，将landscape改为next，保存文件
  2. 再搜索language：，将en改为zh-Hans
  3. 搜索site,可以修改自己的网站标题，作者等等
  4. 在git窗口依次输入以下代码（每次修改博客配置，最好都运行以下代码，下面的设置页面风格也是）
    ‘’’ python
    hexo clean
    hexo g
    hexo s
    hexo d
    ‘’’
  5. 打开浏览器输入博客网址，验证是否成功

设置页面风格

  1. 打开博客文件夹，进入themes文件夹中的next，找到_config.yml，这是主题的配置文件
  2. 找到schemes，可以看到有四行代码，在代码前面输入#字符，则代码不运行，所以这里随便将一行代码前面的#去掉，笔者使用的是Gemini

    注意！每行代码的冒号后面一定要有英文空格，否则会报错，这是学习前端时刻注意的事情

  3. 找到menu,下面有很多行代码（同样去掉#号开启），依次解释一下：
    home：首页
    about:关于（后续可以编写关于自己的简介）
    tags：标签（后续可以为你的文章贴上标签）
    categories：分类（后续可以将你的文章分类）
  4. 找到social，可以设置自己的社交链接，在social_icons代码后面的enable状态改为true
  5. 找到avatar，可以设置自己的头像（需要将头像文件放到博客文件夹里，然后在后面填写路径即可）
  6. 找到wechat_subscriber，可以在每篇文章末尾添加二维码并做解释，同样修改为true，填写路径
  7. 找到reward，可以设置打赏二维码
  8. 找到canvas，有四种动态背景风格，对应的填上true即可
