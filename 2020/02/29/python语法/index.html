矩阵与数组的区别

矩阵[]可以编辑的，数组()是不可以编辑的
例如：
nums = (1,2,3)
nums[0] = 10
nums.append(10)
写出来会报错，数组不支持这种操作

数组和矩阵的简便存储

假如现在有一个数组nothingcanbebetter = (1,2,3),要对数组的各个元素进行运算，传统语言做法是：
x = nothingcanbebetter[0]
y = nothingcanbebetter[1]
z = nothingcanbebetter[2]
对每个元素逐一的赋值，而在python里面可以这样写：
x,y,z = nothingcanbebetter
